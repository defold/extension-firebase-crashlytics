# Firebase Crashlytics for Defold

Defold [native extension](https://www.defold.com/manuals/extensions/) that integrates [Firebase Crashlytics](https://firebase.google.com/docs/crashlytics) functionality on Android, iOS, and macOS.

[Manual, API and setup instructions](https://www.defold.com/extension-firebase-crashlytics/) are available on the official Defold site.
